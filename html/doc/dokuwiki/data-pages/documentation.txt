UML editing
  * free (only free versions considered here)
  * readable textfile
  * text editing
  * mouse positioning (none, absolute, limited)

  * Creately - cannot save file
  * UMLetino - simple XML, no arrow positioning in xml, no undo
  * PlantText - looks like an old copy of PlantUML
  * PlantUML - simple text, positioning is automatic, no way to manually position objects
  * Visual Paradigm - binary file format, unlimited objects
  * draw.io, diagrams.net - complex xml, positioning, free, unlimited

^ Product         ^ xml ^ readable ^ text editing ^ manual positioning ^ arrow redraw ^
